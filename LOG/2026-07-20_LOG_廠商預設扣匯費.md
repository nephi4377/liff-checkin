# 2026-07-20 LOG — 廠商名冊「預設扣匯費」

## 目的

有些廠商接受扣匯費、有些不接受；每筆待匯款再勾太麻煩。改在廠商名冊設預設，新建請款／待匯款自動帶入。

## 改動

| 位置 | 內容 |
|------|------|
| 廠商主檔欄位 | `remit_fee_default`＝`是`／`否`（空視同是） |
| `vendors.html` | 表單下拉＋列表顯示 |
| 請款／待匯款建單 | 依廠商預設寫入 `remit_fee_apply`；單筆仍可在待匯款改 |
| SPEC／資料字典 | 已對齊 |

## 驗收（部署後）

1. 廠商名冊編輯一家 → 預設扣匯費選「否」→ 儲存
2. 用該廠新建待匯款（金額 >30）→ 勾選應為未勾（公司出）
3. 另一家維持「是」→ 新建應為有勾
4. 待匯款頁仍可手動改勾選並儲存
