# 2026-07-10 廠商名冊：新增儲存後明確進入編輯模式

## 問題
新增廠商按儲存後，畫面幾乎不變（欄位仍是剛填的資料），使用者不易察覺「已存成功／已可綁 LINE」。

## 作法
採「跳入編輯模式」（不清空），並加強可見回饋：

| 變更 | 說明 |
|------|------|
| `vendors.html` | 頁內 `feedbackBar` 真正顯示成功／錯誤 |
| 同上 | 標題旁「新增模式／編輯模式」徽章；編輯時表單藍框＋儲存閃綠 |
| 同上 | 按鈕文案：新增＝「儲存」、編輯＝「更新」；出現「取消編輯」 |
| 同上 | 新增成功後捲到 LINE 綁定區 |
| 同上 | 儲存後 `fillForm` 保留待上傳照片，避免背景上傳被清掉 |

## 驗收
1. 新增一筆 → 頂部綠條「已進入編輯模式」；徽章變綠「編輯模式」；按鈕變「更新」；LINE 區出現
2. 按「取消編輯」→ 表單清空、回到新增模式
3. 有拍照再儲存 → 仍會背景上傳（不被清掉）
