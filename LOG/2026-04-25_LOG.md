### 改動項目

**CSS 設計系統升級**
- 導入 `@import url(Google Fonts Inter)`、完整 CSS Custom Properties（HSL 色票、8 個 shadow 層級、radius 規格）。
- 頁面加入深色漸層頂部橫幅（`body::before`），整體由白底升級為 `#f0f4f8` 灰藍底。
- 標籤列（Tab Nav）改為玻璃態 (`backdrop-filter: blur`) 樣式，Active 狀態以白色卡片呈現。
- 工項卡（`item-row`）加入左側 4px 彩色狀態條（`::before`），對應 pending/process/done/cancelled 不同顏色。
- 進度條從純色改為 CSS 漸層，並依完成率動態切換（0%→灰、20%+→橘、60%+→藍、100%→綠）。

**全新組件**
1. **Toast 通知系統**：`showToast(msg, type, duration)` - 右上角帶入場動畫的浮動通知（success/error/info/warning）。
2. **搜尋過濾列**：全文搜尋（工項名稱、備註、分類、區域），即時 `filterItems()` 無需重載。
3. **快速篩選丸**（Filter Pills）：全部 / 未開工 / 施工中 / 完工 / 待審核。
4. **Dashboard 統計摘要**：案場總數、平均進度、未完工數三格 StatCard。
5. **Skeleton loader**：Dashboard 初始載入前顯示骨架佔位動畫。
6. **回到頂部按鈕**：浮動工具列加入向上捲動快捷。

**狀態標籤升級**
- 舊版純文字 `(施工中)` → 新版帶圖示的 `status-chip`（pill 型徽章，不同狀態有對應背景色）。
