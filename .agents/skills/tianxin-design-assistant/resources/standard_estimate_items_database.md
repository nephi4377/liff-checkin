# 添心設計助理知識庫 — 標準裝修估價單之八大核心工項與細節

本文件收錄室內裝修工程中，一般估價單編排的八大核心工項大類、各工項施工細節、材料規範、計價基準，並區分**「新成屋大樓」**與**「中古屋/老屋翻新（舊屋）」**之工項差異，供 AI 助理隨時載入與參考。

---

## 一、 估價單編寫與排版三大原則

1. **依工種與施工順序分章（順序一致性）**
   估價單大類一律以施工的物理先後順序進行排序（如保護最先，清潔收尾最末），嚴禁不同工種（如泥作與木作）項目交錯。
2. **空間分區明細（分區客製化）**
   針對木作工程與系統櫃工程等高客製化項目，應依「玄關、客廳、主臥、次臥一、次臥二」等空間分區列出，以便客戶對照平面配置圖。
3. **規格與備註透明化（防禦性防呆）**
   每個項目必須載明**品牌、厚度、尺寸界線、施工工法**。如「台灣麗仕矽酸鈣板」、「Panasonic星光系列插座」、「系統櫃D60 H240以內」等，避免後續追加爭議。

## 二、 八大核心工項大類及內容細節

### 1. 保護工程（開工第一步）
保護工程是用於保護現有建物公設與室內既有設備在施工期間不受碰撞損傷，並在完工後進行垃圾清理。
*   **室內地坪保護**：預設採用 PP 防潮板打底，上鋪 1.2mm ~ 1.8mm 夾板（雙層保護），全室鋪設，單位以「坪」計價。
*   **室內廚具/衛浴保護**：針對建商附贈之廚具、鋼石檯面、衛浴設備以防塵防刮保護膜及氣泡布包覆。
*   **室內門片與門框保護**：採用防撞泡棉套包裹門框，門片以保護板封板防護。
*   **室外公設保護（管委會規定）**：電梯車廂內部（含壁面、鏡面、地板防護）、梯廳天花/壁面、公共走道地坪鋪設保護板與角防護條，以「式」計價。
*   **粗清與裝潢垃圾清運**：施工期間垃圾堆積清理、裝車、載往合法廢棄物處理場之清運費，以「車」或「式」計價。

### 2. 拆除工程（舊屋翻新/客變調整專屬）
新成屋一般無需此大項，主要用於中古屋翻新或新屋局部格局變更。
*   **舊裝潢木作拆除**：天花板、木地板、舊櫃體拆除與清運。
*   **泥作牆體隔間拆除**：紅磚牆、輕隔間牆或局部局部牆體打除。
*   **地坪/磁磚打石拆除**：浴室防潮地磚、客餐廳舊磁磚打除至見鋼筋打底層。
*   **廚具/衛浴設備拆除**。

### 3. 水電工程（基礎隱蔽工程）
水電為裝修的核心承重牆，涉及用電安全與給排水流暢。
*   **開關/插座新增與移位**：移位與新增出線。通常使用 Panasonic 星光或 COSMO 麗適系列。以「座」或「點」計價（含5米內配線、埋入式配線盒、CD管走線工法）。
*   **高功率專用迴路（110V/220V）**：廚房電器櫃（放烤箱、微波爐、洗碗機等）、冷氣室外機專用迴路。預設採用 5.5mm² 專用絞線拉至總開關箱，以「迴」或「處」計價。
*   **弱電系統延伸（網路/電視）**：Cat6 網路線路移位延伸、電視同軸訊號線路延伸，以「點」或「路」計價。
*   **燈具控制線路與崁燈安裝**：崁燈拉線五米內（一迴路控制上限 10 盞）、吊燈與軌道燈出線移位。
*   **消防灑水頭修改**：大樓高層樓若天花板封板，消防灑水頭需配合天花板高度進行切管、降頭或升頭施作（需由具消防執照工班配合），以「處」計價。

### 4. 泥作工程（舊屋專屬，新屋客變極少用）
涉及結構、防潮防水與貼面美化。
*   **水泥砂漿粉刷打底**：打石後的牆面及地面重新以 1:3 水泥砂漿打底抹平。
*   **浴室/陽台防水施作**：採用彈性水泥，牆面施作高度至少 180cm（濕區頂天），地坪塗佈至少三道防護，以「間」或「坪」計價。
*   **地磚/牆磚鋪貼**：拋光磚、石塑地板打底鋪貼，浴室防滑地磚、壁磚貼面，以「坪」或「平方公尺」計價。
*   **門窗框崁縫**：新設鋁窗或木門框安裝後的水泥填縫防漏施作。

### 5. 木作工程（空間天花板與造型骨架）
主要是木工角材與板材的現場施作。
*   **防潮平釘/造型天花板**：天花板防潮集成角材 (LVL) 骨架 + 台灣麗仕（LUX）矽酸鈣板（防燃一級、無毒低甲醛），以「坪」計價。
*   **間接照明天花板（間照）**：天花板邊緣預留燈槽（層板燈），以「延尺」計價。
*   **冷氣包管與窗簾盒**：冷氣室內機銅管與排水包覆封板、窗戶上方遮擋窗簾軌道之木作窗簾盒，以「尺」或「處」計價。
*   **造型背牆/半腰背牆**：沙發背牆、床頭背牆以木作底板打底，貼皮或油漆，以「尺」計價。
*   **隱藏暗門/木作門框**：與造型牆面融合的無把手隱藏門。

### 6. 系統櫃工程（收納量體的主力）
工廠裁切後現場組裝，相較於木作更環保且耐用。
*   **櫃體類型**：玄關鞋櫃、客廳電視矮櫃、收納高櫃、主臥衣櫃、次臥衣櫃、書桌、化粧桌等。
*   **計價標準**：預設以櫃體「高度 H240cm、深度 D60cm 以內」為延長尺計價標準（例如開門高衣櫃以 5,200 ~ 6,500 元/尺計價）。
*   **收邊與美背**：包含櫃體頂部與側邊貼牆處的系統板现场裁切「速成收邊（打矽利康）」。櫃體背面若露出需加做「美背處理」另計。
*   **內含五金**：標準單價已含進口緩衝鉸鏈、普通三節抽屜軌道；其餘如拉籃、轉角小怪獸、升降衣架等特殊五金需另外拆列報價。

### 7. 油漆工程（美妝面漆與收尾）
決定裝潢最終質感的關鍵工種。
*   **矽酸鈣天花板/牆面油漆**：天花板或牆面接縫處填補 AB 膠（防裂）、批土二至三道（砂磨平整）、面漆二道（預設採用水泥漆，升級乳膠漆另計），以「坪」計價。
*   **冷氣包管/窗簾盒油漆**：針對木作包管的油漆表面處理，以「尺」計價。
*   **木作修補與櫃體牆縫填縫（矽利康收邊）**：木作與系統櫃體、牆面交接處的矽利康填縫修補，以「式」計價（約占木作小計的 5%）。
*   **藝術塗料/特殊漆**：如電視牆清水模塗料、礦物漆，由專人施作，以「式」或「坪」計價。

### 8. 收尾與清潔工程（完工驗收交屋）
完工後的細緻整理與設備安裝。
*   **粗部清潔**：施工期間廢棄物、木屑粗步清理，利於後續工種施作，以「式」計價。
*   **竣工細清**：裝潢後的全室細緻清潔。包含保護板拆除、全室粉塵吸塵、窗戶（含窗溝拆洗）、櫃體內外擦拭、地坪拖洗、殘膠去漆等，以「坪」或「次」計價（須派專業細清公司進場）。
*   **細清後局部修補點漆**：細清或家具搬入後，針對牆面油漆碰撞擦傷的局部補土點漆修補，以「次」計價。

## 三、 新成屋與舊屋之估價單工項差異對照

| 工類大項 | 新成屋大樓估價要點 | 中古屋/老屋翻新估價要點 |
| :--- | :--- | :--- |
| **1. 保護工程** | 重視室外公設保護（因大樓公設新，管委會規定嚴格）；室內保護以地坪與建商附贈廚衛為主。 | 公設防護依規施作；室內保護比重較輕（因室內要大拆除）。 |
| **2. 拆除工程** | 無。僅在極少數客變或格局打牆時有局部拆除。 | **核心大項**。包含舊裝潢、舊地磚、隔間牆、廚衛設備全面拆除，垃圾清運車次多。 |
| **3. 水電工程** | **按需新增/移位**。新成屋管線健全，僅需針對電視牆、電器櫃及床頭書桌進行插座增設（數量通常較少，如 11~19 座）。 | **全室管線重拉（安全核心）**。水管、電線全面抽換更新（安全考量）；插座與迴路全面重新配置。常採空間計價（約 12,000 元/區）。 |
| **4. 泥作工程** | 無。建商已鋪好拋光磚或木地板，浴室廚房皆為成品。 | **核心大項**。浴室地坪牆面重新打底、做防潮防水三道、重新貼磚；客餐廳地坪重新打底。 |
| **5. 木作工程** | 以封冷氣管線、遮光窗簾盒及平釘天花板為主（多為天花板部分）。 | 天花板封板、修復拆除後的裸露牆面、管道間封板，造型牆面比例較高。 |
| **6. 系統櫃工程** | 高度客製化的系統櫃收納為裝修主力。 | 配合木作與格局重整，全面規劃收納。 |
| **7. 油漆工程** | 僅針對天花板矽酸鈣板及局部包管進行批土油漆（牆面多保留建商原漆）。 | **全室重新批土油漆**。因有舊牆面磨損與拆除痕跡，需全室重新整平批土、上漆（工程量極大）。 |
| **8. 收尾與清潔** | 竣工細清（以清除天花板與櫃體木屑粉塵為主）。 | 重度細清（需清除水泥殘渣、重度粉塵、舊污垢等，難度較高）。 |
