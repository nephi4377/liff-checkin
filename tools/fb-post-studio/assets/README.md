# FB 發文工作室 — LOGO 資產

## 為什麼不能直接用 `.ai`？

瀏覽器（以及本頁的 Canvas）**無法讀取 Adobe Illustrator 的 `.ai` 檔**。  
請在 Illustrator 裡先匯出成網頁能吃的格式，再放到這裡或在頁面上「上傳自訂 LOGO」。

## 建議匯出步驟（繁中白話）

1. 用 Illustrator 打開品牌 LOGO 的 `.ai` 檔。  
2. **檔案 → 匯出 → 匯出為…**  
3. 選 **PNG**：
   - 勾選／設定**透明背景**
   - 解析度建議長邊 **≥ 2000px**（發文疊加才夠清晰）
4. 存成 `logo.png`，覆蓋本目錄的同名檔。  
5. （建議再做一次）**檔案 → 匯出 → 匯出為… → SVG**，存成 `logo.svg`。

## 現況

- `logo.svg`：透明底 placeholder（含「添心設計」字樣，僅供開發／預覽疊加位置）。
- `logo.png`：由 placeholder 匯出的透明 PNG；**非正式品牌檔**。

## 在頁面上臨時用

工具內可按「上傳自訂 LOGO」，選 PNG／SVG／WebP／JPG 即可，不必改本目錄檔案。  
若誤選 `.ai`，頁面會提示改匯出格式。

## 原則

- 發文成品的品牌字／LOGO **一律用真實 PNG／SVG Canvas 疊加**
- **不要**要求 Gemini 在圖上畫品牌字樣
