# FB 發文工作室 — LOGO 資產

## 現況

- `logo.svg`：可用的**透明底 placeholder**（含「添心設計」字樣，僅供開發／預覽疊加位置）。
- `logo.png`：由 placeholder 匯出的透明 PNG；**非正式品牌檔**。

## 請設計師處理

品牌正式 LOGO 請從 Illustrator **`.ai` 檔**匯出：

1. 透明背景 **PNG**（建議長邊 ≥ 2000px）→ 覆蓋本目錄 `logo.png`
2. （建議）同步匯出 `logo.svg`

工具內也可在頁面「上傳自訂」臨時 LOGO，不必改檔。

## 原則

- 發文成品的品牌字／LOGO **一律用真實 PNG／SVG Canvas 疊加**
- **不要**要求 Gemini 在圖上畫品牌字樣
