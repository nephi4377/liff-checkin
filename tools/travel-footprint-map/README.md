# 旅遊足跡地圖

互動式世界地圖，載入 Google「已儲存的地點」JSON，以羅賓森或麥卡托投影查看旅遊足跡。

## 一鍵風格（免 API）

| 按鈕 | 效果 |
|------|------|
| **裝飾海報** | 向量陸地 + 羅賓森（配色在「質感配色」換） |
| **街道 Voyager** | Carto 彩色街道圖，好認路 |
| **街道 OSM** | OpenStreetMap 街道圖 |
| **衛星空拍** | Esri 衛星影像 |

## 底圖與配色

| 類型 | 說明 |
|------|------|
| **向量陸地** | 羅賓森／麥卡托，6 種質感配色，陸地與背景對比加強 |
| **Carto / OSM / Esri** | 免金鑰網路圖磚，麥卡托，可看清街道或衛星 |
| **Google 地圖** | 需 API 金鑰（進階設定），本機儲存 |

- **標記大小**：右側滑桿 0.6×～2.5×
- **放大時**：標記變大、向量海岸線變細
- **羅賓森** 僅向量模式；圖磚模式自動用麥卡托

## Google 金鑰怎麼申請

1. 至 [Google Cloud Console](https://console.cloud.google.com/) 建立專案
2. 啟用 **Maps JavaScript API**
3. 建立 API 金鑰，貼到網頁「Google 地圖 API 金鑰」→ 儲存
4. 選底圖「Google 地圖」或「Google 衛星」

## 怎麼開

1. 雙擊 **`開啟.bat`**
2. 瀏覽器會開啟 `http://127.0.0.1:5188`
3. 按 **「載入內建範例」** 或 **「選擇 JSON 檔案」**

> 必須用本機小伺服器開啟（`開啟.bat`），直接雙擊 `index.html` 可能無法載入地圖底圖。

## 資料放哪

| 路徑 | 用途 |
|------|------|
| `data/saved-places.json` | 內建範例（你的 Google 已儲存地點） |
| `assets/ne_50m_land.geojson` | 世界陸地輪廓底圖 |

日後更新足跡：把 Google 匯出的 JSON 覆蓋 `data/saved-places.json`，或在網頁上選新檔案即可。

## 操作說明

- **重點城市**：相近地點合併，適合總覽（預設）
- **全部足跡**：每一筆地點各一個小點
- **羅賓森 / 麥卡托**：切換地圖投影
- **滾輪 + 拖曳**：放大、移動
- **右側國家列**：點一下跳到該區
- **匯出 PNG**：下載目前畫面（含縮放位置）

## 檔案結構

```
travel-footprint-map/
  index.html      主頁
  app.js          地圖邏輯
  styles.css      樣式
  開啟.bat        一鍵啟動
  data/           你的地點 JSON
  assets/         世界地圖底圖
```
