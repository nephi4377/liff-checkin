如何從 Google 匯出？（圖文步驟）

到 Google Takeout 下載「已儲存的地點」，解壓後找到 JSON 再載入本頁。

開啟 Google Takeout ↗
  1. ① 開啟 Takeout 點上方按鈕，登入 Google 帳號
  2. ② 只勾「地圖」 取消全選 → 只留「地圖（你的地點）」
  3. ③ 建立匯出並下載 格式選 ZIP，等 Email 或頁面出現下載連結
  4. ④ 解壓並載入 路徑通常為 Takeout/地圖/已儲存的地點.json，用上方「選擇 JSON 檔案」
