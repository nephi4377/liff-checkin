一、記現場進度（#752 這種）

在內部 LINE 群、且帳號有開權限時，用新的一行以 # 加案號開頭，把進度記到主控台該案的日誌。

正確 
#752
今天上午泥作收工
明天貼磚
換案 
#752
752 進度…

#880
880 進度…
  • # 後面直接打數字（1～4 位），中間不要打「案號」兩字。
  • 案號那一行下面幾行都算同一案，直到下一行又出現 #數字 才換案。
  • #752 標題 或 #777陳 這種緊接文字也可以。
  • 成功時 LINE 會回 Flex 卡片，可點按鈕開啟該案工作區。

二、照片怎麼傳（3 分鐘內）

先傳圖或先打案號都可以，系統會在 3 分鐘 內把照片併進日誌。

先傳圖 → 再打案號

照片會併入第一個案（同則訊息裡最上面的 # 案號區塊）。

（先傳 1～多張照片）
#752
（可只打案號，或加文字）

先打案號 → 再傳圖

照片會併入最後一個案（同則訊息裡最後一個 # 案號區塊）。

#752
泥作收工
（3 分鐘內再傳照片）

同則寫兩案時

先傳的圖 → 併第一案（例如 #752）。
後傳的圖（案號文字已送出之後）→ 併最後一案（例如 #880）。 

（先傳照片）
#752
752 進度
#880
880 進度
（之後再傳的照片 → 併 880 案）
  • 超過 3 分鐘的照片不會自動併入上一則；請再打一次案號，或改為先傳圖再打案號。
  • 照片不會跨人：A 傳的圖不會併進 B 打的案號。
  • 只有文字、完全沒照片時，同日同案可能併成同一則日誌；有照片一律新開一則。

三、寫進度 vs 綁客戶（別搞混）

寫法 用途 範例
寫進度日誌
#752		 記現場進度到主控台工作區 
#752
泥作完成
綁定客戶
#案號 752		 把 LINE 客戶／群組綁到案號（設計圖自動分類） #案號 752

若打成 #案號 752，走的是客戶綁定，不會變成現場進度日誌。

四、指令一覽

在 LINE 輸入 #help 可取得精簡版與本頁連結。

現場與主控台

#752 …（行首 #＋1～4 位數字）
內部群寫進度日誌，見上文第一、二節。
#主控台
開啟整合主控台。
#專案 [案號]
開啟該案工作區。例：#專案 752；不填案號則為待審核日誌。
#案場資訊 [關鍵字]
查案場詳細資訊、複製進場資料。例：#案場資訊 752
#新增案場
開啟表單建立新案場資料。

客戶／群組

#案號 [3～6 位數字]
綁定客戶與案號。例：#案號 752
#客戶 [關鍵字]
模糊搜尋客戶。例：#客戶 YUN
#設定 [ID] [案號]
為指定 ID 設定案號。例：#設定 U123456 752

註冊

#員工報到
員工註冊身分並啟用選單。
#廠商報到
廠商註冊身分並啟用選單。

說明

#help
在 LINE 顯示精簡說明與「查看完整指令說明」按鈕（連到本頁）。

五、常見問題

Q：打了 #752 沒反應？

請確認：(1) 在內部群 (2) 帳號有白名單權限 (3) 格式是行首 #752 不是 #案號 752。

Q：照片沒併進去？

是否超過 3 分鐘？請重打案號或先傳圖。多案同則時，確認照片是「先傳（第一案）」還是「後傳（最後一案）」。

Q：空區塊沒寫入？

若某案區塊完全沒文字、也沒有可併照片，系統會提示該案未記錄；請在 # 案號後至少打一點內容，或先傳照片再打案號。
