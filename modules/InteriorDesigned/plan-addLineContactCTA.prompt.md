# 添加 LINE 聯繫鼓勵訊息計畫

## 目標
在 LayoutPlanner.html 頁面中新增多個位置的鼓勵使用者與添心設計聯繫的訊息，並包含 LINE 官方帳號 `@uis9604v`。

## 實施位置

### 1. 手機版提示訊息 (`#mobile-warning`)
- **目標**：在現有的手機版警告視窗中新增 LINE 聯繫按鈕
- **動作**：
  - 在「添心設計」品牌名稱下方新增 LINE 聯繫區塊
  - 提供可點擊的 LINE 官方帳號按鈕，顏色使用 LINE 綠色 (#00B900 或 #06C755)
  - 顯示 LINE 官方帳號 `@uis9604v`

### 2. 主畫面右下角浮動按鈕區 (`#minimized-bar`)
- **目標**：在現有的浮動按鈕區新增「聯繫我們」按鈕
- **動作**：
  - 在工具箱和預算按鈕旁邊新增「📞 聯繫設計師」或「💬 LINE 諮詢」按鈕
  - 採用綠色配色（LINE 品牌色），使其視覺上突出
  - 點擊時開啟 LINE 官方帳號或顯示聯繫資訊模態框

### 3. 預算明細 Modal 視窗 (`#budget-modal`)
- **目標**：在預算明細視窗底部新增專業諮詢呼籲
- **動作**：
  - 在「匯出 CSV」和「另存 PDF」按鈕下方新增「諮詢詳細報價」區塊
  - 包含文案鼓勵使用者與設計師聯繫進行個性化報價
  - 提供 LINE 聯繫選項（按鈕或 QR 碼）

### 4. JavaScript 事件處理
- **目標**：實現 LINE 聯繫功能
- **動作**：
  - 為所有「聯繫」按鈕新增點擊事件監聽器
  - 處理 LINE 開啟邏輯（應用程式或網頁版本）
  - 備用方案：如果無法直接開啟 LINE，顯示 LINE ID 供手動複製

## 技術細節

### LINE 連結格式
1. **應用程式深層連結**：`line://ti/@uis9604v`
2. **網頁版本**（備用）：`https://lin.ee/...` (需建立短連結)
3. **複製 LINE ID**：提供簡單的複製按鈕，讓使用者可手動複製 `@uis9604v`

### 樣式設計
- **LINE 品牌色**：#06C755 (官方綠色) 或 #00B900
- **按鈕樣式**：參考現有的 `toggle-btn` 或 `.toggle-btn` 樣式
- **文案**：鼓勵性語氣，強調免費諮詢、專業設計師等價值主張

## 優先順序
1. **高優先**：手機版提示 + 右下角浮動按鈕（最顯眼）
2. **中優先**：預算明細 Modal 視窗（轉換關鍵時刻）
3. **低優先**：額外的 QR 碼或詳細聯繫資訊（視需要）

## 考量事項
- **使用者體驗**：不應過度打擾，避免訊息疲勞
- **版本相容性**：確保 LINE 連結在各種設備和瀏覽器上都能正常運作
- **備用方案**：提供複製 LINE ID 的選項，以應對連結無法開啟的情況
