# Google Sheets 元件資料庫使用說明

## 設定步驟

### 1. 建立 Google Sheets 試算表

在 Google Sheets 建立一個新試算表,**工作表名稱必須是「工作表1」**,格式如下:

| 名稱 | 寬度 | 深度 | 價格 | 圖片網址 |
|------|------|------|------|----------|
| 三人沙發 | 210 | 90 | 35000 | https://... |
| L型沙發 | 280 | 170 | 55000 | https://... |
| 電視櫃 | 240 | 45 | 18000 | https://... |

**欄位說明**:
- **名稱**: 元件名稱 (必填)
- **寬度**: 寬度(cm) (必填)
- **深度**: 深度(cm) (必填)
- **價格**: 價格(元) (選填,預設為 0)
- **圖片網址**: SVG 或圖片網址 (選填,留空會使用預設圖示)

> **重要**: 工作表名稱必須是「工作表1」(Google Sheets 預設名稱)

### 2. 設定試算表為公開

1. 點擊右上角「共用」按鈕
2. 點擊「變更」連結
3. 選擇「知道連結的任何人」
4. 權限設為「檢視者」
5. 點擊「完成」

### 3. 取得試算表 ID

從試算表網址中複製 ID:
```
https://docs.google.com/spreadsheets/d/【這段就是ID】/edit
```

例如:
```
https://docs.google.com/spreadsheets/d/1ABC123xyz456/edit
```
ID 就是: `1ABC123xyz456`

### 4. 在應用程式中載入

1. 開啟 Layout Planner
2. 在左側工具箱找到「Google Sheets 元件庫」區塊
3. 貼上試算表 ID
4. 點擊「載入元件」按鈕

## 注意事項

- ✅ 試算表必須設為**公開**才能載入
- ✅ **工作表名稱必須是「工作表1」**(這是 Google Sheets 的預設名稱)
- ✅ 第一行為標題行,會自動跳過
- ✅ 圖片網址可使用 `data:image/svg+xml,...` 格式的 SVG
- ✅ 留空圖片網址會自動產生帶文字的預設圖示
- ✅ 每次點擊「載入元件」會覆蓋之前載入的 Google Sheets 元件
- ⚠️ 從本地檔案開啟時,使用 Google Visualization API 繞過 CORS 限制

## 快捷鍵

- **R** = 旋轉選中的元件
- **D** = 刪除選中的元件 (不需確認)
- **方向鍵** = 移動選中的元件 (按住 Shift 加速)
- **拖曳** = 移動元件
- **右下角控制點** = 調整元件大小
