# 添心室內裝修設計 - 數位工具開發總結 (Implementation Summary)

## 專案概況
本文件彙整「添心室內裝修設計」所有數位工具的開發細節、技術架構與功能說明。目前專案包含三大核心模組，旨在透過網頁技術提升室內設計的規劃效率、現場管理與客戶溝通體驗。

## 1. 互動式室內設計規劃工具 (LayoutPlanner)
**檔案**：`LayoutPlanner.html`, `LayoutPlanner.js`
**版本**：v9.1 (Auto-save & Crash Prevention)

### 1.1 核心功能詳解

#### A. 智慧排版系統
- **拖放排版 (Drag & Drop)**：直覺拖曳家具元件，支援吸附對齊。
- **方向性拉伸 (Directional Resizing v9.0)**：
    - 支援 **上(N)、下(S)、左(W)、右(E)** 四向獨立拉伸。
    - **中心點補償**：拉伸時會自動計算中心點位移，確保物件不會「飄移」，而是像真實物體一樣單側延伸。
- **鍵盤微調 (Nudge)**：支援使用方向鍵進行像素級 (1px) 微調，按住 `Shift` 可加速 (10px)。
- **碰撞檢測 (Collision)**：移動或調整尺寸時，若與其他物件重疊會顯示紅色警示框，防止配置錯誤。

#### B. 專業繪圖工具
- **多邊形空間繪製**：點擊畫布自定義不規則房間形狀，支援天花板與地板分層繪製。
- **自動坪數計算**：系統根據繪製的多邊形面積，自動換算為台灣常用的「坪數」單位 (採無條件進位制)。
- **磁吸頂點**：繪製新區域時，游標會自動吸附至現有區域的頂點，確保無縫拼接。

#### C. 安全與備份 (v9.1)
- **即時自動儲存**：
    - 機制：每 60 秒背景自動執行 `LocalStorage` 備份。
    - 範圍：包含家具位置、尺寸、旋轉角度、自定義繪圖區及底圖設定。
- **防崩潰還原**：若瀏覽器異常關閉，重新開啟時會自動偵測未儲存的備份，並跳出「是否還原？」提示。

#### D. 報價與輸出
- **動態計價引擎**：
    - 支援多種計價模式：固定單價 (Fixed)、按寬度 (Width)、按深度 (Depth)、按面積 (Area)。
    - 即時聯動：調整元件尺寸時，右側屬性面板的報價會同步更新。
- **PDF 報價單**：一鍵生成專業報價單，包含各項工程 (天花、地板、保護) 小計與總預算。
- **圖片匯出**：生成帶有浮水印的設計圖 (JPG)，方便透過 LINE 分享。

### 1.2 操作使用手冊 (User Manual)

#### 基礎操作
1.  **載入底圖**：點擊左側「上傳底圖」按鈕，選擇建設公司提供的平面圖 (JPG/PNG)。
2.  **設定比例**：使用「底圖縮放」滑桿調整底圖大小，使其符合格線比例 (建議以門寬 90cm 為基準)。
3.  **放置家具**：從左側「元件庫」拖曳家具至畫布。
4.  **調整視角**：
    - **縮放畫布**：按住 `Ctrl` + 滾輪，或使用右下角 Zoom 按鈕。
    - **移動底圖**：勾選「底圖調整模式」後，可自由拖曳底圖位置。

#### 進階編輯
- **旋轉物件**：選取物件後，點擊屬性面板的「↻ 旋轉」按鈕 (每次 90 度)。
- **複製物件**：選取後按 `Ctrl + D` 或點擊面板上的「複製」按鈕。
- **刪除物件**：選取後按 `Delete` 或 `Backspace` 鍵。
- **復原/重做**：使用 `Ctrl + Z` (復原) 與 `Ctrl + Y` (重做)，或點擊介面上的對應按鈕。

#### 快速鍵列表 (Hotkeys)
| 按鍵組合 | 功能 |
| :--- | :--- |
| **Ctrl + Z** | 復原 (Undo) |
| **Ctrl + Y** | 重做 (Redo) |
| **Ctrl + D** | 複製選取物件 (Duplicate) |
| **Delete / Backspace** | 刪除選取物件 |
| **↑ / ↓ / ← / →** | 微調位置 (1px) |
| **Shift + 方向鍵** | 快速移動 (10px) |
| **Enter** | (繪圖模式中) 完成繪製 |
| **Esc** | 取消繪圖 / 取消選取 |

### 1.3 外部依賴與技術
- **html2canvas**: `1.4.1` - 用於畫布截圖與浮水印合成。
- **jspdf**: `2.5.1` - 用於生成 A4 格式報價單。
- **jszip**: `3.10.1` - 用於專案封存打包。
- **Google Sheets API**: 用於動態載入家具元件資料庫與價格表。

## 2. 施工日誌生成器 (Site Report Generator)
**檔案**：`site-report.html`
**版本**：v11.1 (Multi-page Support)

### 2.1 核心功能
專為手機端優化的現場回報工具，解決「照片零散、缺乏整理」的痛點。
- **手機優先介面 (Mobile First)**：大按鈕、觸控友善，適合現場人員單手操作。
- **智慧拼圖 (Smart Layout)**：根據上傳照片數量 (1~6張)，自動切換最佳版型 (全版、對開、田字、六宮格)。
- **自動分頁 (Auto Pagination)**：支援無限量照片上傳，每 4 張自動生成新的一頁 (Page 1/N)，保持圖片清晰度。
- **一鍵生成**：前端即時合成高解析度 JPG (1200x1600)，壓印公司 Logo、日期、天氣與工項備註。

### 2.2 技術實現細節
- **React + Tailwind**：使用 React 管理 UI 狀態 (照片陣列、表單資訊)，Tailwind 快速構建響應式介面。
- **Client-side Canvas**：完全不依賴後端，所有圖片合成邏輯皆在瀏覽器端透過 `canvas.getContext('2d')` 完成。
- **圖片處理**：使用 `FileReader` 預覽圖片，並實作 `drawImage` 的 `cover` 裁切模式 (類似 CSS `object-fit: cover`)，確保照片填滿格子不變形。

## 3. 照片轉正工具 (Floorplan Scanner)
**檔案**：`floorplan-straightener.html`
**版本**：v1.0.24

### 3.1 核心功能
解決業主提供「歪斜書面平面圖」的問題。
- **透視校正**：使用者拖曳四個角落定位點，系統將梯形圖片拉直為矩形平面圖。
- **自動偵測**：利用電腦視覺演算法自動尋找圖紙邊緣。
- **無變形裁切**：支援僅裁切多餘背景而不進行透視變形。

### 3.2 技術實現細節
- **OpenCV.js**：引入 WebAssembly 版的 OpenCV 進行圖像矩陣運算 (`cv.getPerspectiveTransform`, `cv.warpPerspective`)。
- **放大鏡輔助**：實作 `Magnifier` 元件，在拖曳角落時顯示局部放大圖，提升選點精準度。
