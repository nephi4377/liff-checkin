## 1. 產品定位與目標使用者

- **定位**：CODING 核心設計引擎；單頁式室內平面**互動排版**，結合 Google 試算表驅動之**元件庫與單價**，產出**視覺平面圖**與**粗估報價**（含免責聲明）。主程式 `LP_LayoutPlanner.js` 為大型幾何／計價模組（Canvas 2D 空間配置與動態決策）。
- **目標使用者**：需快速在平面底圖上配置櫃体／家具、標註工程項、圈選天花／地板施工區並試算費用之內部或客戶端使用情境。
- **明確非目標**：非專業 CAD／非即時協作編輯、非經正式簽核之工程契約報價。

## 2. 執行環境與技術架構

### 2.1 執行環境

- **建議**：以本機或網站 **HTTP(S) 伺服器**開啟；**不建議**以 `file://` 直接開啟（ES `import`、部分 API 行為可能異常）。
- **模組**：`LP_LayoutPlanner.html` 以 `<script type="module" src="./LP_LayoutPlanner.js">` 載入，依賴同目錄之 `LP_core.js`（通知、SVG 消毒、幾何／碰撞輔助函式等）。
- **行動裝置**：以 **User-Agent 含 `Mobi`** 判斷為行動裝置時，顯示全畫面提示並**中止主程式初始化**（避免小螢幕操作）；曾修正 iframe 內寬度誤判，改以 UA 為主。

### 2.2 前端技術棧（CDN）

- **Tailwind CSS**（`cdn.tailwindcss.com`）
- **html2canvas**：畫布匯出 PNG、ZIP 內截圖
- **JSZip**：打包 PNG + JSON + CSV
- **已移除**：先前曾載入 **OpenCV.js** 但未於本工具呼叫；已自 `LP_LayoutPlanner.html` 移除以減少體積。

### 2.3 遠端資料來源

- **Google Sheets**：透過 **gviz** 公開 JSON（`tqx=out:json&gid=0`）讀取元件清單。
- **試算表 ID**（寫死於 `LP_LayoutPlanner.js`）：`1y8iD3Pe8AvYxDYFGYVOZ0afsdW10j1GSnDXqUCyEh-Q`
- **LP_LayoutSheetViewer** 使用相同 ID 與 `gid=0`，僅供預覽／編輯 I 欄文字，**不寫回** Sheet。

### 2.4 座標、渲染與核心引擎

**三級座標**

1. **畫布座標 (Canvas space)**：滑鼠事件觸發的直接位置。
2. **世界座標 (World space)**：考慮 `viewScale`（畫布縮放比）後的相對座標；所有 Drag／Click 須除以 `viewScale` 還原真實座標。
3. **物理尺寸 (Physical space)**：透過 `pixelsPerMeter` 換算實際單位；`1 尺 = 30 cm`，計價無條件進位至 **0.5 尺**。

**三層渲染**

1. **Drawn Areas Layer (SVG)**：地板、天花板、牆體填充；`fill-rule: evenodd` 內孔扣除。
2. **Placed Cabinets (DOM)**：家具以 DOM 包覆 SVG，用 CSS `transform` 處理旋轉與鏡像。
3. **Drawing Preview／Interaction Layer**：頂點控點、選取框。

**碰撞與牆體（`LP_core.js`）**

- **SAT**：分離軸定理，支援旋轉矩形；**MTV** 最小推離位移實現硬碰撞與貼齊；**AllowOverlap** 可由 Sheet M 欄豁免（地攤、標註等）。
- **牆壁引擎**：`offsetPolygon` 外擴路徑；`evenodd` 組合內外圈模擬厚度；**Ortho Snap**（Shift／Alt）正交鎖定。
- **牆體鎖定**：分離「繪圖鎖定」與「牆體鎖定」，佈置家具時不誤觸結構。
- **SVG 校正**：`autoFixSvgGeometry` 偵測 StrokeWidth 向內縮進，避免裁切；載入時經 `normalizeSheetSvg`（消毒 + 校正）。

> [!IMPORTANT]
> 修改渲染或互動時，務必確保 `viewScale` 全域變數同步更新，以免滑鼠選取偏位。

## 3. 功能規格（現況行為）

### 3.1 工作區與檢視

- **畫布尺寸**：固定 **2000×2000**（cm 邏輯空間；標頭顯示 2000cm×2000cm）。
- **網格**：50px 間距（`#grid-layer`）。
- **縮放**：`viewScale`；**Ctrl + 滾輪**、縮放按鈕、重置；套用於 `#design-canvas` transform。
- **捲動**：`#canvas-wrapper` 捲動；載入後嘗試捲至畫布中心。

### 3.2 底圖（平面圖）

- **上傳**：本機圖檔 → `data:` URL 顯示於 `#bg-img`。
- **底圖調整模式**：勾選後可調整縮放；未勾選時控制區 `opacity-50 pointer-events-none`。
- **底圖變換**：獨立 `bgScale`／`bgPosition`，支援平移縮放「照圖描繪」。
- **比例**：建議搭配「比例尺 90CM」元件與門寬比對；屬操作指引，非自動校正。

### 3.3 元件庫（櫃体／家具等）

- **載入**：自 Sheet 依 **群組（J 欄）** 分類，渲染於「元件」頁籤。
- **拖放**：拖入畫布建立 `placedCabinets`（位置、旋轉、寬深、透明度、備註、才數、鏡像、副屬性等）。
- **互動**：點選、拖曳、四向縮放、**R** 旋轉、**M** 鏡像、**[`]** 圖層、方向鍵微移、**D／Delete** 刪除、**Ctrl+D** 複製。
- **碰撞**：SAT 對 OBB 重疊偵測；拖曳以 MTV 推開；邊緣貼齊不視為碰撞（見 §5）。
- **手動改寬深**：`applyCabinetVisualUpdate` 僅更新單一 DOM，避免整批重繪失焦。
- **圖像來源**：I 欄為內嵌 SVG 或 http(s) URL；經 `sanitizeSvgString`、`normalizeSheetSvg`、`processSvgStr` 處理。

### 3.4 區域繪製（天花板／地板／牆壁）

- **模式**：按鈕進入；**點擊**加頂點；**Enter** 完成；**Esc** 取消。
- **天花／地板**：多邊形面積 → 坪數（面積 ÷ 30000 再 `Math.ceil` 為 `areaInPing`）。
- **地板損耗**：`calculateFloorAreaWithLoss`：×1.2 後進位至 **0.5 坪**。
- **連結元件**：完成後連結 Sheet「平釘天花板」「超耐磨木地板」列（若存在），否則程式內建預設。
- **牆壁**：至少 3 點；`calculateWallPath` 厚度偏移；**牆壁不列入 `calculateFullQuotation`**（`type === 'wall'` return）。
- **圖層開關**：可切換天花／地板／牆壁顯示；可鎖定牆壁／繪圖。

### 3.5 工程標註

- **新增**：畫布中央建立文字方塊 + 指示點（預設如「隱藏門」）。
- **計價**：可走主項目 + 副屬性；`source === 'annotation'` 且 `customAddons[0]` 可覆寫單價／數量／單位。

### 3.6 預算與報價

- **總價**：`updateQuotation` → `calculateFullQuotation` 之 `grandTotal`。
- **明細 Modal**：依工程分組插入標題列。
- **合併列規則**（`calculateFullQuotation`、`quotationAssignSpatialClusterKeys`）：
  - **帳面條件相同**才比對：群組、名稱、單價、單位、`pricingType`、備註、副屬性指紋等。**不以**櫃寬高或區塊坪數決定是否分列。
  - 同批項目以外接矩形算**邊到邊距離**；距離 ≤ **55**（畫布座標）視為相鄰，union-find 成 **spatialCluster**。
  - **僅同一 spatialCluster** 合併成報價同一列；不相鄰則分列，相鄰則合併（寬高／坪數可不同）。
  - 僅一筆可計價項目不帶位置群後綴。
  - **規格**欄：多種規格以「、」串接。
- **分組排序**：內建 `preferredOrder`（保護 → 拆除 → … → 清潔）；實際群組取決於 Sheet／標註資料。
- **匯出**：列印／PDF（`@media print` + `#budget-modal`）、CSV、ZIP 內 CSV。
- **免責聲明**：Modal 內建僅供參考文字。

### 3.7 輸出與檔案

| 動作 | 內容 |
|------|------|
| **另存圖片 (PNG)** | html2canvas 截取 `#design-canvas`，可選浮水印 |

### 3.8 復原／重做與自動備份

- **歷史**：`history`／`historyIndex`，最多約 **50** 步（`MAX_HISTORY_STATES`）；**Ctrl+Z**／**Ctrl+Y**／**Ctrl+Shift+Z**。
- **自動儲存**：每 **60 秒** 寫入 `localStorage`（`layoutPlanner_autosave_v1`）；啟動時 24 小時內有備份則 `confirm` 詢問還原。

## 5. 計價與碰撞（摘要）

**計價**

- `calculatePrice`：`width`／`depth` 線性、`area`／`cai` 面積、`fixed` 固定；副屬性經 `addonsConfig` 擴展。
- `cmToFeet`：1 尺 = 30 cm，進位 0.5 尺。
- `calculateFullQuotation`：彙總元件、區域、標註後依 §3.6 合併；**總價為 0 不列入**。

**碰撞**

- SAT 四頂點；`overlap` 嚴格大於 + epsilon，邊貼邊不算碰撞。
- 拖曳以 MTV 修正；卡住顯示 `collision-warning`。
- `mirrored`：`getVertices` 交換局部四角再旋轉，與 SAT 一致。
- 拉邊縮放若 `checkCollision` 為真則還原縮放前尺寸。
