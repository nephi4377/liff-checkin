3.1 路徑 A：錢已經發生（立刻進帳）

收支登錄／群組記帳／表單 → 立刻寫收支明細 → 有案號就帶毛利。

  S0["開始：收支登錄／群組／表單"] --> S1["確認是誰、權限"]
  S1 --> S2["整理成標準欄位"]
  S2 --> S3{"欄位齊全？"}
  S3 -->|否| S3a["回覆缺什麼"]
  S3 -->|是| S4["寫進收支明細（該月）"]
  S4 --> S5["寫完後自動連動"]
  S5 --> S6["有照片 → 存檔＋附件索引"]
  S5 --> S7["有案號 → 毛利明細_X00"]
  S5 --> S8["收入＋案號 → 客戶收款待確認"]
  S5 --> S9["操作稽核"]

3.2 路徑 B：錢還沒付（請款 → 匯款才進帳）

送請款時不寫收支；標記已匯款時才寫帳、有案號才進毛利。

  T2 --> T3["主管核准／退回"]
  T3 -->|退回| T2
  T3 -->|核准| T4["待匯款清單"]

  T5 -->|"產中信檔"| T6["只產檔，還不算已匯"]
  T5 -->|"標記已匯款"| T7["這時才寫收支明細"]
  T6 -.->|"匯完再標"| T7
  T7 --> T8["有案號 → 毛利"]
  T7 --> T9["可通知廠商／記附件"]

8. 資料本一覽

試算表多本；檔案（照片／合約）多在雲端硬碟，表上存連結。

資料本放什麼誰常寫
收支明細已發生收支（按月）路徑 A 立刻；B 匯款結案
會計主檔廠商、請款、分攤、薪資請款名冊／請款／審核／待匯
案件毛利總覽、明細_X00、客戶財務分頁記帳連動、手動、客戶確認
單據附件索引帳與照片對應有上傳時
薪資相關審核與轉帳薪資流程
