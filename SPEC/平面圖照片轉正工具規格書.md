# 平面圖「照片轉正」工具 — 白話說明（給非工程背景）

**這是什麼？**  
用手機或相機拍的平面圖，常常歪歪的、有透視。這個網頁工具可以幫你把圖「拉正」，或先裁掉多餘的邊再繼續調。

**檔案在哪？**  
`modules/InteriorDesigned/floorplan-straightener.html`  
從公司系統選單進去時，網址會類似：`#/floorplan-straightener`（在 `spa/app.js` 裡有登記）。

---

## 一般人怎麼用（照順序）

1. **等畫面說引擎好了**  
   第一次開要連網下載一些程式庫；沒載好之前不能選照片，這是正常的。

2. **上傳一張圖**  
   支援常見照片格式（例如 JPG、PNG）。

3. **（可選）按「自動偵測」**  
   電腦會試著猜圖紙的四個角。若猜不準，可以多按幾次切換別組猜測結果。  
   **畫面上**：候選虛線**預設不顯示**（避免干擾拖黃點）；勾選「顯示偵測候選」後以**極淡灰／淡琥珀**虛線對照（可選「稍清楚」）。**橘色實線＋黃點**＝正在編輯的邊界。自動偵測結果已**收斂至多約 6 組**。

### 自動偵測（白話原理，對應程式）

1. 把圖縮到寬不超過約 1200 像素再算（減輕負擔）。  
2. 轉成灰階 → 稍微模糊。  
3. 用**多種二值化**產生黑白線條圖後各自找輪廓：**適應性二值化（正／反）**、**Otsu（正／反）**、以及 **Canny 邊緣＋膨脹**（利於拼貼／透視合成圖）。  
4. 對夠大的輪廓：先嘗試「簡化成剛好四角」；也對**全圖面積最大的輪廓**加強嘗試（**漸進簡化**、**凸包再簡化**、**最小外接斜長方形**）。面積門檻約 **2.5%～10%** 分段使用。  
5. **拼貼、透視合成、情境照**通常沒有「單一紙張外框」，自動結果可能仍不準；若仍無任何候選，程式會加上**貼齊整張圖邊**的一組四角當備援，方便使用者手動拖點。  

因此若圖裡還有大片門框、地磚、陰影邊，也可能被當成「大框框」而多出幾組候選；**不保證**一次就對準你的圖紙外框，需靠多次切換或手動拖點。

4. **拖四個黃點**  
   把點對準圖紙真正的四個角。拖的時候會出現小放大鏡，方便對準。  
   （窄螢幕或畫布被縮小時，程式會把手指／滑鼠位置換算成畫布內部座標，避免點對不準。）

5. **「應用裁切（無變形）」**  
   用一個**不歪、不拉透視**的方式，把四個點**外圍的長方形**裁下來，換成新的底圖。  
   **白話重點**：若你的四個點是斜的，裁出來的長方形**邊還是直的**，所以角落可能會多帶一點背景——這是刻意設計成「不要讓圖被拉歪」。

6. **「執行轉正」**  
   這一步才會做**透視校正**（把斜的圖變成方正的圖）。完成後可以下載 PNG。  
   若這一步或「應用裁切」失敗，畫面上會出現**白話說明**（不是只有工程用主控台），可按「知道了」關閉。

7. **不滿意**  
   可以「重新調整」回到上一步，或「重新上傳」換一張圖。  
   （「重新調整」後若曾出現空白畫布，v1.0.26 起已修正：外框寬度還沒算出時會先用螢幕寬推算，版面變動時也會自動重畫。）

## 使用時要知道的限制

- **一定要能連網**（用瀏覽器開）：圖形引擎、介面樣式是從網路上載入的；網路不穩可能載很久或失敗。
- **圖在你自己的電腦／手機上處理**：沒有把圖上傳到我們公司的伺服器做運算；但若你擔心隱私，仍請勿在公共電腦留下下載檔案。
- **很舊的瀏覽器**可能跑不動；建議用常見的新版 Chrome／Edge／Safari。
