# 施工圖審核 — 參考重點（知識庫）

|------|------|
| 性質 | **知識庫 / 判斷原則**，不含 PROMPT 指令 |
| 用途 | 供三步驟 PROMPT 於 User 訊息末段附帶，避免 AI 使用通用常識誤答 |

> 本檔為**公司內部慣例彙整**，數值僅作建議基準；**以現場與設計決策為準**。

---

## 1. 圖面共通要求

- **圖框基本資訊**：案名、日期、圖號（公司內部**不強制** Revision 與完整追溯）。  
- **比例尺與方位**：每張圖有比例尺、北向（或主入口方向）。  
- **單位與基準**：全案統一 mm 或 cm；明確標示「結構面 / 完成面」。  
- **圖例／符號表**：插座、開關、燈具、檢修口、材質代號**皆需對應圖例**。  
- **尺寸閉合**：各分段尺寸加總＝總長；崁入件兩側**留現場公差**（Asobi）。

## 2. 空間尺寸慣例（建議基準）

| 項目 | 建議數值 |
|------|----------|
| 主要動線淨寬 | ≥ 80–90 cm |
| 廚房走道（單排） | ≥ 90 cm |
| 廚房走道（雙排） | ≥ 100–120 cm |
| 掛衣櫃淨深 | 55–60 cm |
| 書架／展示櫃淨深 | 25–40 cm |
| 鞋櫃淨深（鞋子橫放） | ≥ 35 cm |
| 窗簾盒淨深（單層） | ≥ 15 cm |
| 窗簾盒淨深（雙層） | ≥ 25 cm |
| 樓梯淨高 | ≥ 200 cm |

## 3. 水電重點

### 3.1 水電明細表（審水電圖時**必附**）

建議欄位：迴路編號、位置、用途、型式（插座／開關／專用）、安培、對應設備、高度（EL）、備註。

### 3.2 插座／開關高度（EL）常用基準

| 位置 | 建議高度（完成面） |
|------|--------------------|
| 一般插座 | EL+30 cm |
| 床頭插座 | EL+60–75 cm（避開床墊與背板） |
| 書桌／梳妝 | EL+75–90 cm |
| 流理台上方 | EL+110–120 cm |
| 洗衣機 | EL+110 cm 以上（避免積水） |
| 冰箱／大電器 | EL+180 cm（或依機型） |
| 開關 | EL+110–120 cm |

### 3.3 專用迴路建議項

廚房三機（烘烤蒸）／IH／洗碗機／熱水器／暖風機／資訊櫃／智慧居家主機／冷氣／電動窗簾控制。

### 3.4 冷氣

- 冷媒管長度越短越直越佳；**排水有坡度**、終點明確（排至陽台／排水孔）。  
- 室外機：**散熱、檢修空間、噪音**同時考量。  
- 室內機：避開床頭、沙發正面直吹。

---

## 4. 立面／櫃體重點

- **壁板／造型牆**：必須標示**厚度／完成面**，慣用如 **+100** 表示完成面加厚（實際以圖例為準）。  
- **美背、踢腳、側封**：標示**施作深度**與和地坪／壁面關係；避免與踢腳線、插座衝突。  
- **櫃體俯視**：與立面搭配，驗證淨深是否符合功能。  
- **材質交接**：異材質拼接處必須有收邊策略（壓條、對斜、齊平）。  
- **門片／五金**：開啟方向、滑軌型式、鉸鏈角度、拉籃預留。

---

## 5. 材質與估價一致性

- 板材色號、石材型號、塗料品牌**必須與估價清單 100% 一致**。  
- 圖上**可一眼對應到施作位置**；僅見於表單但圖面未指位者視為不明確。  
- 收邊條規格（材質、寬度、顏色）應一併標示。

---

## 6. 衛浴

- **防水上翻**：淋浴區 ≥ 180 cm；其他濕區 ≥ 30 cm 以上（依設計）。  
- **洩水方向**：以地漏為最低點，乾濕分離落差處注意不絆腳。  
- **排糞管**：馬桶中心與排糞管對位，標示管徑與管中心距牆。  
- **檢修口**：浴缸／淋浴拉門軌道／天花排風機上方。

---

## 7. 廚房

- **三角動線**（冰箱、水槽、爐口）盡量互不干擾。  
- 抽油煙機**排風管長度越短越直越佳**，避免多重轉折。  
- 水槽下排水高度、淨水器、濾水器預留。  
- IH / 瓦斯選型與對應迴路、排氣。

---

## 8. 天花與照明

- 標示**淨高（CH）**、造型起點座標、燈位、空調出回風口。  
- **維修口尺寸與位置**、**間接照明手部操作距離**。  
- 燈具建議表欄位：位置、型號、色溫（K）、瓦數（W）、口徑、調光、迴路編號。

---

## 9. 常見踩雷清單（自檢用）

- [ ] 水電明細表**有附**且與圖面對得上  
- [ ] 床頭、沙發、餐桌、書桌**插座高度**有避開家具背板  
- [ ] 冷氣排水**終點**畫清楚、坡度足夠  
- [ ] 壁板／造型牆**厚度**（+100 等）語意明確  
- [ ] 美背／踢腳**施作深度**有標  
- [ ] 櫃體淨深在俯視圖上可驗證  
- [ ] 重點材質**圖上可一眼對位**，且與估價一致  
- [ ] 天花檢修口、弱電箱、熱水器**留得出維修動線**  
- [ ] 異材質交接有收邊策略  
- [ ] 單位（mm / cm）與基準（結構面／完成面）**全案一致**
